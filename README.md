Hoe speel je het zelf?
1. Download de nieuwste versie van Node.js van <https://nodejs.org/en/download>
2. Download de .zip met alle code ([downloadlink](<https://github.com/qastrol/zwaksteschakel/archive/refs/heads/main.zip>)), en pak de .zip uit in een nieuwe map
3. Open start_server.bat
4. Als het goed is start dit bestand een lokale server op. Bij De Zwakste Schakel open je de twee links die in de opdrachtprompt te zien zijn (waarschijnlijk http://localhost:3000/dashboard en http://localhost:3000/display of een ip-adres).
5. Via de dashboardpagina of index.html bedien je als "presentator" het scherm dat voor de toeschouwers te zien is via display.html
6. Als je OBS gebruikt, maak je een browserbron aan met http://localhost:3000/display als URL. Als je via Discord speelt kan je gewoon een browser openen en de browser streamen naar Discord. Je kan ook twee beeldschermen gebruiken als je het spel offline op locatie wilt spelen met iedereen in dezelfde kamer.
7. Veel speelplezier!
